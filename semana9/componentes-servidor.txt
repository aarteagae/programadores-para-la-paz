
Sistema operativo: Es el software principal que gestiona el hardware de una computadora y permite que otros programas funcionen. Actúa como el intermediario entre los componentes físicos, el usuario y las aplicaciones instaladas.

Node.js: Es un entorno de ejecución que permite utilizar el lenguaje JavaScript fuera de un navegador web, directamente en el sistema operativo. Se emplea principalmente para construir herramientas de línea de comandos y servidores backend eficientes.

Aplicación server.js: Es el archivo de código fuente que define la lógica y el comportamiento de un servidor web. Contiene las instrucciones exactas sobre qué rutas existen (como /reportes o /estado) y qué debe responder el sistema ante cada petición.

Puerto: Es un canal de comunicación o "puerta virtual" específica dentro de una computadora (por ejemplo, el puerto 3000). Sirve para direccionar el tráfico de red de internet directamente hacia la aplicación correspondiente, evitando conflictos con otros programas.

Logs: Son el registro cronológico de eventos e historial de actividad que un sistema genera mientras está en ejecución. Permiten monitorear qué peticiones han ingresado, detectar fallos en tiempo real y auditar el comportamiento del servidor.
